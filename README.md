# Kidzkapers

Kidzkapers is a modern event booking and celebration web application created for a startup that provides birthday surprise and party planning services in Nagpur.

The platform helps users explore and book celebration services with a clean and responsive user interface.

---

# 🎉 Features

* Modern responsive UI
* Event and celebration showcase
* Dynamic content management using Sanity CMS
* Fast frontend deployment with Netlify
* Service listing and presentation
* Interactive user experience
* Mobile-friendly design

---

# 🛠 Technologies Used

## Frontend

* React.js
* JavaScript
* CSS

## Backend / CMS

* Sanity CMS

## Deployment

* Netlify



